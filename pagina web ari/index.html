Conoce un poco de ARI

ARI es un robot creado con Arduino diseñado para entrar a lugares de difícil acceso para el ser humano o demasiado complicado para otro tipo de maquinarias. Es un diseño adaptable en tamaño y componentes, lo que permite personalizarlo según el ambiente y la tarea.

  • Creado con Arduino Nano
  • Cámara integrada para exploración y rescate
  • APK para control y visualización
